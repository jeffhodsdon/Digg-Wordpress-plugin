The official Digg plugin.  Adds Digg buttons to posts and a sidebar widget. 

== Description ==

The official Digg plugin adds buttons and widgets to your wordpress blog.  The following is a breakdown of the features.

Digg buttons:
*   Options to have all the different size buttons on each post
*   Controls for changing the position of the button in the context of a post
*   Custom CSS to the div containing the button
*   Default topic for the submission if the entry has yet to be submitted

== Installation ==

To install the Digg plugin follow these steps.

1. Upload the contents of the plugin archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= The widget is not appearing on my blog =

After generating your widget through the generator you must enable it from the widgets section of the admin panel.
